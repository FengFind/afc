The SeT GIS Library (also SeTGIS) is a library that provides a set of utilities to manage geo-referenced data structures (GIS stands for Geographical Information System).

SeTGIS was introduced in 2004 with the Metro-B application. In 2008, with the release of the version 2.0.

Features

SeTGIS is composed of several modules described in the following table.

Each class has a tag which is indicating its Maven artifact and group identifiers.

ArtifactGroupDescription
setgisdatatransferfr.utbm.set.gis Tools to transfer GIS elements via clipboard, drag 'n drop...
setgisdisplayfr.utbm.set.gis Platform-independent implementation of a GIS element drawer (independent from AWT, SWING and SWT).
setgiselementfr.utbm.set.gis Geo-referenced shapes: polygon, polyline...
setgisfwfr.utbm.set.gis Application middleware for Geographical Information System.
setgisgraphiccontextfr.utbm.set.gis SWING graphic context to draw GIS elements.
setgislayerfr.utbm.set.gis Container of GIS elements organized as layers.
setgislayerpanelfr.utbm.set.gis SWING component which is displaying GIS elements (from inside a GIS layer).
setgislayertreefr.utbm.set.gis SWING tree which is displaying the hierarchical structure of GIS layers.
setgislocationfr.utbm.set.gis Geo-localized point and area
setgisprimitivefr.utbm.set.gis Geo-referenced abstract primitive types.
setgispropertypanelfr.utbm.set.gis SWING panel to display and input the properties of GIS elements.
setgisroadfr.utbm.set.gis Road network and roads.
setgisroadlayerfr.utbm.set.gis GIS Layer dedicated to road networks.
setgisroadpathfr.utbm.set.gis Path of roads.
setgisroadprimitivefr.utbm.set.gis Abstract types dedicated to road network and roads.
setgisshapefilefr.utbm.set.gis ESRI Shape file reader which is able to output GIS elements.
